In order to compile YUMI exFAT in NSIS, you'll need to create a YUMI2USB.7z file and place it here containing the following parts, inside;

> YUMI2USB.exe compiled as release win32 using Visual Studio from the source files found in YUMI2USB.zip.
> The boot and ventoy folders from Ventoy-1.0.88-windows.zip compiled using CentOS to include filestructure changes found in YUMI-THEMED-zip.
